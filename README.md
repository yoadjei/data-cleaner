# Data Cleaner

A web application for cleaning CSV datasets. Built with Streamlit.

## Features

Upload CSV files and apply cleaning operations including duplicate removal, missing value imputation (mean, median, mode, KNN), outlier detection, type conversion, and text normalization. Supports light and dark themes.

## Installation

```
pip install -r requirements.txt
```

## Usage

```
streamlit run app.py
```

Then open http://localhost:8501 in your browser.

## Output

Download cleaned CSV files and cleaning logs directly from the interface.
